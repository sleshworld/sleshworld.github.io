# Каталог лекций Kinescope

Приложение загружает каталог по адресу:

`https://sleshworld.github.io/lectures/catalog.json`

## Структура папок

В одном проекте Kinescope создаются папки курсов, а внутри них папки разделов:

```text
Лекторий 3.0
├── Лекция сегодня
├── Важное
└── Июль (записи)

Пойми Кукуха 6
├── Доктор Бутузов
└── Психолог Воробьёв
```

Готовое видео из папки раздела автоматически попадает в соответствующий раздел приложения. Название видео в Kinescope становится названием лекции.

Соответствие папок и идентификаторов приложения хранится в `kinescope-map.json`.

## Настройка GitHub

В репозитории откройте `Settings → Secrets and variables → Actions` и добавьте:

- `KINESCOPE_API_TOKEN` — отдельный API-токен из настроек Kinescope;
- `KINESCOPE_PROJECT_ID` — идентификатор проекта с папками курсов.

После этого откройте `Actions → Sync Kinescope lectures → Run workflow`. Далее каталог проверяется автоматически каждые 30 минут.

Токен Kinescope нельзя добавлять в `catalog.json`, код приложения или Git-репозиторий.
